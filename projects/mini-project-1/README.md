# CodeMastery - Educational Web Platform

## Project Overview

CodeMastery is a comprehensive, modern educational platform designed to teach programming and web development through structured learning paths, real-world projects, and interactive resources. Built with **HTML5, CSS3, and Bootstrap 5**, this project demonstrates clean web design, semantic HTML, and responsive layouts.

## What's Included

### Pages

- **index.html** - Homepage with hero section, featured courses, learning paths, and platform overview
- **courses.html** - Catalog of programming courses across different difficulty levels
- **learning-paths.html** - Structured learning roadmaps (Frontend, Backend, Full Stack, etc.)
- **projects.html** - Portfolio of real-world project case studies with technical breakdowns
- **resources.html** - Curated library of docs, tools, templates, and guides for developers
- **topics.html** - Comprehensive algorithms and data structures reference with code snippets
- **blog.html** - Blog section for technical articles and learning updates

### Styling

- **style.css** - Custom CSS with:
  - CSS variables for theming (purple, gold, gray palette)
  - Gradient backgrounds and glowing animations
  - Responsive grid and flexbox layouts
  - Smooth transitions and hover effects
  - Bootstrap 5 integration

### Assets

- **images/** - Gallery folder with project mockup images and promotional visuals

## Customization

Modify the design by updating CSS variables in `style.css`:

```css
:root {
  --purple: #7c3aed;      /* Primary color */
  --gold: #ffd700;         /* Accent color */
  --light-gray: #e5e7eb;   /* Borders & backgrounds */
  --font-body: 'Inter';    /* Body font */
  --font-brand: 'Poppins'; /* Heading font */
}
```
